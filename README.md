# MyPetDiary

## サービス概要
ペットに関する日々の記録を管理するアプリです。
ペットは複数登録可能で、過去の記録を一覧表示することで
ついつい忘れがちな定期的なお世話の次の予定をチェックすることができます。

## このサービスへの思い・作りたい理由
私は犬＆猫の多頭飼いをしているのですが、それぞれお世話の内容やタイミングが異なるので、
1つのアプリに記録が集約できると便利だと思い開発することにしました。

## 想定されるユーザー層
ペットを飼われているすべての方。
特に、多頭飼いでペットごとに異なるお世話（投薬、通院、お手入れ等）を管理したい方。

## サービスの利用イメージ
投薬や通院以外に、お出かけやお手入れなどの記録として使ってもらいたいです。
ログ種別を選択しての記録が可能で、種別は自由に追加登録できます。
また、ログを記録する際は体重も併せて登録でき、体重のみの登録も可能です。
複数のペットを登録できるので、それらを一覧で確認することができます。

## ユーザーの獲得について
Xで紹介、今後の改善ポイントとして検索エンジン対策。

## サービスの差別化ポイント・推しポイント
* ペット複数登録を可能にして多頭飼いに対応（登録数制限なし）
* カテゴリを選んでボタンを押すだけの簡単登録
  - このアプリに向いている方・・・何かイベントがあった時だけ登録して一覧で振り返りたい方
  - このアプリに向いていない方・・・毎日体調などを細かく記録して管理したい方
* ログ記録は、日付のみor日時を選べます

## 実装を予定している機能
### MVP
* ユーザー登録
* ログイン機能
* ペットの基本情報登録
* 日々のログ記録
  - お世話・お出かけなどのログを記録（種別を選択して登録）
  - 体重の記録（ログとセット、または単独で記録可能）
* 記録の閲覧機能（過去の活動と体重の変化を確認）
* ログ種別のカスタマイズ（デフォルトの種別に加えて、自分専用の種別を作成）

### その後の追加予定機能
* 体重グラフ表示
* お世話予定リマインダー機能
* ログ共有機能
